# Machine Learning Project

This project builds and evaluates machine learning models using:
- Data preprocessing
- Feature engineering
- Model training
- Hyperparameter tuning
- Explainable AI (SHAP, LIME)

## Files
- Notebook (.ipynb)
- Dataset
- Model file (.pkl)
- requirements.txt

## How to Run
1. Install packages: `pip install -r requirements.txt`
2. Open the notebook
3. Run all cells
